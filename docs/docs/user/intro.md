# User Guide

This section is for people using Nebula as an application.

It will progressively cover the user-facing workflow:

- installation and first launch
- project creation and basic navigation
- graph editing features and everyday usage
- exporting, generation and common user tasks
- troubleshooting for typical usage issues

If you are looking for contributor setup or internal technical details, use the Developers section instead.
