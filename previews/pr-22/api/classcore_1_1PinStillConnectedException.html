Detailed Description

Exception thrown when attempting to modify pin metadata while pins are still connected.

This exception is thrown when a node needs to edit its own pins metadata (such as changing pin count, types, or structure) but one or more pins still have active connections. All pins must be disconnected before modifying the node's pin configuration.
