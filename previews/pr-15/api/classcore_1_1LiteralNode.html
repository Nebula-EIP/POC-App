Validates if this node's output pin can connect to a target node's input pin.

This method should be called from the source node to check if a connection is valid. To check if a node can receive a connection, call this method from the source node.

Parameters
out_pinThe output pin index on this node.
targetPointer to the target node.
target_in_pinThe input pin index on the target node.
Returns
An expected containing void on success, or an error message on failure.

Implements core::NodeBase.

std::any core::LiteralNode::data ( ) const

Retrieves the data stored in the literal node.

This function returns the data encapsulated within the literal node as a std::any object. The actual type of the data can be determined by the caller using std::any_cast along with LiteralNode::type().
