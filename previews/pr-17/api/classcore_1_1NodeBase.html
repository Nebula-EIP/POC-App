Detailed Description

Abstract base class for all nodes in the graph system.

NodeBase provides the common interface and functionality for all node types. Nodes can only be created and managed by the Graph class through the factory pattern.

Adds a new input pin to this node.

Allocates a unique pin id and appends the pin to the input pin list. Should be called from InitializeConnections() or by the Graph when managing dynamic pin layouts (e.g. function parameters).

Note: These methods are protected to ensure only the node itself and the Graph (friend class) can modify pin topology. External callers must use Graph APIs to manage links/topology.
