Detailed Description

A node representing the return value inside the function's body graph.

Each FunctionOutputNode has exactly one input pin whose data type matches the function's return type.

Validates if this node's output pin can connect to a target node's input pin.

This method should be called from the source node to check if a connection is valid. To check if a node can receive a connection, call this method from the source node.

Parameters
