Detailed Description

Abstract base class for all nodes in the graph system.

NodeBase provides the common interface and functionality for all node types. Nodes can only be created and managed by the Graph class through the factory pattern.

Factory method to deserialize a node from JSON.

Creates the appropriate node type based on the "kind" field in JSON, then calls its virtual Deserialize method to initialize it. The JSON must contain "id" and "kind" fields at a minimum.
