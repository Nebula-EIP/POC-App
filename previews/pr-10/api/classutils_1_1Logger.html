Detailed Description

Thread-safe singleton logger with formatted output and source location tracking.

The Logger class provides a centralized logging system with multiple severity levels, automatic timestamp generation, and source location information. It uses C++20 std::format for message formatting and is thread-safe.

Example usage:

LOG_INFO("Application started");
LOG_ERROR("Failed to load file: {}", filename);
