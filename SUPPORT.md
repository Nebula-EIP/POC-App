# Support

## Getting Help

For usage or development questions:

- Check the documentation in `docs/`
- Search existing pull requests
- Ask in the team's private communication channel

## Current Support Model

GitHub Issues are currently disabled for this repository during the POC phase.

For non-security support requests, use the internal communication channel and include:

- What you expected
- What happened
- Steps to reproduce
- OS and environment details
- Relevant logs or screenshots

## Future Public Issue Tracking

When public issue tracking is enabled again, this document will be updated with issue-reporting guidelines.

## Security Issues

For security-related reports, follow `SECURITY.md` instead of opening a public issue.
